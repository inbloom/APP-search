inBloom Search 1.1

User Guide

  • welcome

    Welcome to inBloom Search Version 1.1. inBloom Search provides an efficient search tool to find educational resources contained in the Learning Registry. This user guide provides you with detailed instructions on the use of this tool.

  • learn about inBloom Search

    inBloom Search provides an efficient search tool to help you find educational resources in the Learning Registry. Using inBoom Search is simple: enter a topic or subject keyword and then click Search. You can then use the filters to quickly narrow the results by criteria such as end user or age range, publisher of the resource, or the Common Core State Standards that the resources support.

  • search for a resource

    inBloom Search finds educational resources stored in the Learning Registry. The Learning Registry is a joint effort of the Department of Education and Department of Defense to create a system for sharing metadata about learning resources, ratings, reviews, comments and annotations, usage information and information about relationships between resources.

    To search for a resource, first enter a keyword in the Search box and then filter your results by criteria (such as educational use or media type), publisher, or Common Core Standard Dot Notation.
