inBloom Search 1.2

User Guide

  • welcome

    Welcome to inBloom Search Version 1.2. inBloom Search provides an efficient search tool to find educational resources that have been cataloged in the Learning Registry Index. This user guide provides detailed instructions on the use of this tool.

  • learn about inBloom Search

    inBloom Search provides an efficient search tool to help you find educational resources that have been cataloged in the Learning Registry Index. Using inBloom Search is simple: enter a topic or subject keyword and then click Search. You can then use the filters to quickly narrow the results by criteria such as end user or age range, publisher of the resource, or the Common Core State Standards with which the resources are associated.

  • search for a resource

    inBloom Search finds educational resources cataloged in the Learning Registry Index. The Learning Registry is a joint effort of the Department of Education and Department of Defense to create a system for sharing metadata about learning resources, ratings, reviews, comments and annotations, usage information, and information about relationships between resources.

    To search for a resource, first enter a keyword in the Search box and then filter the results by criteria (such as educational use or media type), publisher, or Common Core Standard Dot Notation.
