search for a resource

inBloom Search finds educational resources cataloged in the Learning Registry Index. The Learning Registry is a joint effort of the Department of Education and Department of Defense to create a system for sharing metadata about learning resources, ratings, reviews, comments and annotations, usage information, and information about relationships between resources.

To search for a resource, first enter a keyword in the Search box and then filter the results by criteria (such as educational use or media type), publisher, or Common Core Standard Dot Notation.

1. Enter a keyword in the Search box

In the Search box, type the topic or subject of the resource for which you are searching.

If you enter multiple words in the Search box, the system will return results based on relevance. For example, if you entered the following search terms:

triangle median

the results will display in the following order:

  1. The first results returned will match exactly “triangle median”
  2. The next results returned match both “triangle” and “median” (but not necessarily together)
  3. The next results returned will match either “triangle” or “median”

2. Click Search

Click Search. The search results display in the results pane. Web site results display a thumbnail and link to the web site. Other types of resources will display a thumbnail of the resource if available.

If more than 10 results are available, click the Next button at the bottom of the screen to scroll through the results.

(Optional) The default number of results to display per page is 10. To change the number of results per page, select a different number of results from the drop-down list at the right of the Search button.

3. Filter your search results (optional)

There are three ways to filter the results returned from the keyword search:

  • Filter by criteria (intended end user role, typical age range, educational use, interactivity type, learning resource type, media type, or group type)
  • Filter by the Common Core State Standard that the learning resources support
  • Filter by publisher of the resource

One filter or a combination of several filters may be applied. More information on each of these methods is provided below.

Filter the results by criteria

To filter results by a criterion such as age range or intended user, click on the category in the filter list. A side menu will display. Select the filters to apply. More than one filter can be applied. The applied filter(s) display(s) in the list on the left side of the screen. The available filters are:

  • Intended End User Role. The individual or group for which the resource was produced (such as student, teacher, or administrator).
  • Typical Age Range. The typical range(s) of ages for the content’s intended end user.
  • Educational Use. How the resource will be used in an educational setting.
  • Interactivity Type. The predominant mode of learning supported by the learning resource.
  • Learning Resource Type. Instructional format(s) in which the resource is made available to the intended end user.
  • Media Type. Media format(s) in which the resource or component is published or produced.
  • Group Type. The type of group for which this resource is designed (such as inidividual, class, or small group).

Filter the results by publisher

To filter the results by the publisher of the learning resource, in the Publisher field, enter the name of the publisher for the resource. Press the Enter key.

Filter the results by Common Core Standards dot notation

Many of the resources in the Learning Registry are aligned with Common Core State Standards. To narrow results to learning resources that support a specific Common Core State Standard, in the Dot Notation field, enter the identifier of the standard in dot notation format. As you begin typing, possible matches will display in the drop-down box beneath the field. Select the correct standard.

For more information on the Common Core State Standards or to find the dot notation identifier of a specific standard, visit the Common Core State Standard web site using the CCSS Web button at the top of the window.

4. Open the resource

Zoom

Click the linked name to open the resource. The resource opens in a new window.

Optional: Remove a criterion filter

Click the empty set symbol beside the filter to be removed.
