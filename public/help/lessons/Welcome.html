Welcome

Welcome to Search Version 1.0.1 The Search application provides an efficient search tool to find educational resources contained in the Learning Registry. This user guide provides you with detailed instructions on the use of this tool.

Following is overview information to provide context to how Search fits into the Shared Learning Infrastructure.

The Learning Resource Metadata Initiative is a joint venture between Creative Commons and the Association of Educational Publishers to drive the creation and adoption of a metadata schema to consistently describe educational resources.

The Shared Learning Collaborative is a state led initiative funded by the Bill & Melinda Gates Foundation and the Carnegie Corporation designed to develop internet scale infrastructure to make personalized learning a reality for every US student by improving the usefulness, variety and affordability of education technology.

The Learning Registry is joint effort of the Department of Education and Department of Defense to create a system that is designed to share metadata about learning resources, ratings, reviews, comments and annotations, usage information and information about relationships between resources.

This user guide applies to the 1.0.1 release of Search. This version of Search works in the most current versions of Google Chrome and Firefox on Mac and Windows. It should work well in Safari. Future releases will be tested across additional browser and OS combinations.
