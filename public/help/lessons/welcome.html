welcome

Welcome to inBloom Search Version 1.2. inBloom Search provides an efficient search tool to find educational resources that have been cataloged in the Learning Registry Index. This user guide provides detailed instructions on the use of this tool.

About inBloom

inBloom is a state led initiative funded by the Bill & Melinda Gates Foundation and the Carnegie Corporation, designed to develop internet scale infrastructure to make personalized learning a reality for every US student by improving the usefulness, variety and affordability of education technology.

Related Efforts

inBloom is working in parallel with other important efforts, including the Learning Resources Metadata Initiative (LRMI) and the Learning Registry, to ensure metadata standards are in place that allow the vision of standard-aligned educational content to be achieved in a consistent manner across all applications and uses.

The Learning Resource Metadata Initiative is a joint venture between Creative Commons and the Association of Educational Publishers to drive the creation and adoption of a metadata schema to consistently describe educational resources.

The Learning Registry is a joint effort of the Department of Education and Department of Defense to create a system that is designed to share metadata about learning resources, ratings, reviews, comments and annotations, usage information, and information about relationships between resources.

About this Guide

This user guide applies to the 1.2 release of inBloom Search. This version of inBloom Search works in the most current versions of Google Chrome and Firefox on Mac and Windows. It should work well in Safari. Future releases will be tested across additional browser and OS combinations.
