learn about inBloom Search

inBloom Search provides an efficient search tool to help you find educational resources in the Learning Registry. Using inBoom Search is simple: enter a topic or subject keyword and then click Search. You can then use the filters to quickly narrow the results by criteria such as end user or age range, publisher of the resource, or the Common Core State Standards that the resources support.

Zoom

The inBloom Search Window has three panes.

  • The Header bar provides the topic/subject search field and Search button, access to the help system and to a reporting tool to report issues, and a link to the Common Core Standards web site.
  • The Filter pane provides filters for narrowing search results by criteria, publisher, and Common Core Standard dot notation.
  • The Results pane displays the results from the search, including a link to the resource and a thumbnail of the web site or resource.
