Search for a Resource

In the Search box, type the topic or subject of the resource for which you are searching.

Click Search. The search results display in the results pane. A thumbnail and description of the resource display in the results list. If more than 10 results are available, click the Next button at the bottom of the screen to scroll through the results.

Click the linked name to open the resource. The resource opens in a new window.

Optional: Filter the results by criteria

To filter your results, click on a filter category. A side menu will display. Select the filters to apply. You may apply more than one filter. The applied filter(s) display(s) in the list on the left side of the screen. The available filters are:

  • Intended End User Role. The individual or group for which the resource was produced (such as student, teacher, or administrator).
  • Typical Age Range. The typical range(s) of ages for the content’s intended end user.
  • Educational Use. How the resource will be used in an educational setting.
  • Interactivity Type. The predominant mode of learning supported by the learning resource.
  • Learning Resource Type. Instructional format(s) in which the resource is made available to the intended end user.
  • Media Type. Media format(s) in which the resource or component is published or produced.
  • Group Type. The type of group for which this resource is designed (such as inidividual, class, or small group).

Optional: Filter the results by Common Core Standards dot notation

In the Dot Notation field, enter the standard in dot notation format. The only standard currently supported is the Common Core Standard. As you begin typing, possible matches will display in the drop-down box beneath the field. Select the correct standard.

Optional: Filter the results by publisher

In the Publisher field, enter the name of the publisher for the resource. Press the Enter key.

Optional: Remove a criteria filter

Click the empty set symbol beside the filter to be removed.
