Search Window

Zoom

The Search Window has three panes.

  • The Header bar provides the topic/subject search field and Search button, access to a reporting tool to report issues, and a link to the Common Core Standards web site.
  • The Filter pane provides filters for narrowing search results by criteria, Common Core Standard dot notation, and publisher.
  • The Results pane displays the results from the search, including a thumbnail of the web site (if a web site) and a link to the resource.
